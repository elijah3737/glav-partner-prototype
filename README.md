# Glav — Партнёрская программа (прототип)

Кликабельный прототип партнёрской программы «Глав»:
- **Экран клиента** — лендинг акции (телефон + код партнёра → скидка).
- **Кабинет партнёра** — код, баланс, вывод денег, лиды со статусами из CRM.

Стиль повторяет сайт glav.ru (шрифт Mulish, синий #227CD2, светлый серо-голубой фон, скруглённые карточки).

Открыть: `index.html` (self-contained, без сборки). Демо-данные — вымышленные.
